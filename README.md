# Sales Reporting Dashboard

## A relay compliant server built using
- Node
- Express
- Postgres
- GraphQL
- Docker

## Setup

### Setup and run locally using 

```
./setup-local.sh
```

### Build and run docker container locally

```
docker-compose down
docker-compose build
docker-compose up
```

### Develop locally

```
yarn start:local
```

### Start dev server

```
yarn dev
```

### Start prod server

```
yarn prod
```

### Build the application

```
yarn build
```
